About This Project

Real-time stress and emotion detection using facial expressions and speech analysis — powered by deep learning.

What We Do

👁️

Face Emotion Detection

Real-time webcam analysis using CNN model trained on FER2013 dataset. Detects 7 core emotions with 82% accuracy.

🎤

Speech Emotion Detection

Voice recording analysis using LSTM model trained on RAVDESS dataset. Identifies 8 emotional states from speech patterns.

🧠

Stress Analysis

Combines facial and vocal cues to calculate stress score. Provides Low, Medium, or High stress classification.
